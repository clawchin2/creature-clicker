# MONETIZATION AGENT - MEMORY

## ROLE
Maximize revenue without killing retention. Design ethical monetization that feels fair but converts.

## CORE PRINCIPLES
1. Convenience sells better than power
2. Impulse purchases during frustration = highest conversion
3. Whales (1%) generate 50%+ revenue
4. Minnows need to feel progress without paying
5. Time-limited offers create urgency

## PRICING TIERS (Robux → INR)

### Impulse (₹19-39 / 25-49 Robux)
- 2x coins for 10 mins
- Single spin/reroll
- Skip 5 minutes of wait

### Small (₹79-99 / 99-124 Robux)
- Auto-clicker for 1 hour
- Guaranteed Rare+ drop
- 2x coins for 1 hour

### Medium (₹199-249 / 249-299 Robux)
- Exclusive pet/skin
- Auto-hatch 10 eggs
- 2x coins for 24 hours

### Large (₹499-999 / 599-1249 Robux)
- Gamepass (permanent 2x)
- Legendary pet guarantee
- Premium currency pack

### Whale (₹2000+ / 2499+ Robux)
- Limited edition items
- Exclusive titles
- Max-tier pets

## CONVERSION FUNNEL

### Target Metrics
- DAU: 1,000 → 5,000 → 25,000 → 100,000
- Conversion Rate: 2% → 3% → 4% → 5%
- ARPPU: ₹200 → ₹300 → ₹400 → ₹500
- Revenue: ₹4,000 → ₹45,000 → ₹400,000 → ₹2,500,000/month

### Funnel Stages
1. **Install** → Tutorial (retain 80%)
2. **First Session** → First purchase hook (convert 5%)
3. **Day 1** → Daily rewards (retain 40%)
4. **Day 7** → Battle pass/seasonal (convert 10%)
5. **Day 30** → Whale identification (top 1% spend ₹2000+)

## PSYCHOLOGY TRIGGERS

### Frustration Points (High Conversion)
- Running out of energy/coins
- Failing a challenge
- Waiting for timer
- Getting common instead of rare
- Losing progress

### Happiness Points (Retention)
- First legendary drop
- Completing collection
- Beating high score
- Social recognition
- Visual upgrade

### Hook Timing
- First purchase offer: 5 minutes in
- Second offer: First failure
- Third offer: End of first session
- Daily: Login screen
- Weekly: Weekend events

## ETHICAL BOUNDARIES

### Do
- Make game playable without paying
- Offer value in every purchase
- Be transparent about odds
- Limit daily spending (optional)
- Avoid dark patterns

### Don't
- Block progression behind paywall
- Use confusing currency exchange
- Target children with predatory tactics
- Hide real costs
- Create gambling addiction
